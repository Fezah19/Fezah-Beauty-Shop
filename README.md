# Fezah Beauty Shop — Glow your way

Curated beauty products and practical guidance for real routines in Ongata Rongai.

## Live demo
After publishing GitHub Pages:
https://fezah19.github.io/fezah-beauty-shop/

## Structure
- `index.html` — main page
- `styles.css` — branding and layout
- `script.js` — interactions
- `assets/` — product images

## Publish on GitHub Pages
1. Create a repo named `fezah-beauty-shop`.
2. Add all files and commit to the `main` branch.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set **Source** to `Deploy from a branch`.
5. Set **Branch** to `main` and **Folder** to `/root`.
6. Save—wait ~1–2 minutes.
7. Your site will be live at: `https://fezah19.github.io/fezah-beauty-shop/`.

## Customize
- Update WhatsApp link in the header (`wa.me/2547XXXXXXXX`).
- Replace product images in `assets/`.
- Edit prices and descriptions in `index.html`.
